    Launch The Second RocketMQ Code Marathon

    less than 1 minute read

    ROCKETMQ CODE MARATHON

    March 29th 2017

    Objective:

    1. Foster Community
    2. Prosper Ecology
    3. Incubate Committers

    Output:

    1. 1~3 Milestone Releases Per Project

    Rewards:

    1. Two Potential Committers Candidates Per Project
    2. Free Offers to Active Participants for On-Site Meetup

      PROJECTS

    RocketMQ-Big Data

    1. Integration of big data analysis framework
    2. Rich Documentation and Good Test Cases Coverage
    3. Apache Beam preferred
    4. Join by Creating Pull Requests against RocketMQ Beam Repository
    5. Join by Creating Pull Requests against RocketMQ Flink Repository
    6. Join by Existing Pull Request against RocketMQ Storm Repository
    7. Join by Existing Pull Request against RocketMQ Spark Repository

    RocketMQ-Client SDK

    1. Development of RocketMQ Client
    2. Rich Documentation and Good Test Cases Coverage
    3. Go preferred
    4. Join by Creating Pull Requests against RocketMQ Go Repository
    5. Join by Creating Pull Requests against RocketMQ .Net Repository
    6. Join by Creating Pull Requests against RocketMQ PHP Repository

    RocketMQ-Data Replication

    1. Implement Data Replication from Database
    2. Rich Documentation and Good Test Cases Coverage
    3. MySQL preferred
    4. Join by Creating Pull Requests against RocketMQ MySQL Repository
    5. Join by Creating Pull Requests against RocketMQ Redis Repository
    6. Join by Creating Pull Requests against RocketMQ MongoDB Repository
